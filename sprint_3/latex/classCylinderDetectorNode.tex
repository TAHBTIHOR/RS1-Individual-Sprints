\doxysubsection{Detailed Description}
A ROS 2 node that detects cylindrical objects using laser scans. 

This node subscribes to laser scan data, processes the scans to detect cylindrical objects of a known diameter, publishes markers for visualization in RViz, and broadcasts the detected object\textquotesingle{}s position as a transform.
